== Description ==
The SM News Tickr is a wordpress plugin to show post as Jquery News Ticker. It is a very nice Ticker. jQuery News Ticker used to show the letest or importent post or news to site visitor.

= Features =
* Settings Page to set News Ticker options.
* You can select post type to show as News Ticker.
* You can set News Ticker Title.
* Can set  numbe of post to show on News Ticker.
* News Ticker controll show or hide.

== Installation ==
1. You can install the SM News Ticker form wordpress.org plugin or download this plugin and extract is to your wordpress site plugin folder.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Now SM News Plugin activate to your wordpress site.

= Usage =

1. Go to SM News Tickr form left navigation menu.
2. Set all the option for your News Ticker and save it.
3. Set News Ticker shortcode to any place of your page.
5. Hit the save button.
